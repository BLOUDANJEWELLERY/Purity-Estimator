How to Measure Properly (Important!):

• Weigh the gold object dry.
• Place a water-filled container on the scale and zero it.
• Suspend the object using a thin thread or wire so it doesn’t touch the container.
• Submerge it fully — record the increase in weight shown (that’s the displaced water weight in grams = cm³).
• Avoid hand contact with water or letting object rest on container.
